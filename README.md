# digital-design-journey

This is where I'm documenting everything I learn about digital design — RTL, FPGA, and eventually processor design. Projects go from simple combinational logic all the way up to FSMs, counters, and hopefully RISC-V down the road.

Everything is written in SystemVerilog, simulated in Questa, and synthesized with Vivado.
